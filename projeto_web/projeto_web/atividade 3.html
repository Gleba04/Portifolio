Formas geométricas

Círculo
Um círculo é uma figura plana formada por todos os pontos que estão a uma distância fixa de um ponto central
Triângulo
Um triângulo é uma figura geométrica plana com três lados e três ângulos
Retângulo
Um retângulo é uma figura geométrica plana com quatro lados e quatro ângulos retos (90 graus)
